For windows 


Traditional Chinese
-- 
批次PDF上鎖保全, 
步驟:
0. 請先確認電腦是否安裝 java 11 或更新的版本
1. 選擇未上鎖PDF所在目錄(該目錄可多個PDF)
2. 設定PDF上鎖保全後, 匯出之目錄
3. 設定開檔案的密碼(可不設定)
4. 設定保全密碼(一定要設定)
5. 執行成功後,可在步驟2設定目錄找到上保全檔案

English
--
batch prevent PDF from printing& copy,
Step:
0. Java 11 or above are required
1. select source directory that PDF files
2. setup destination directory that exported PDF files
3. setup password that open pdf file (optional)
4. setup password that can unlock protected pdf
5. click Execute , expoted pdf files that can be found at step 2 destination directory
   
InMethodProtectPDF.xml is configuration file used by launch4j to convert runnable jar file to windows  execute file
